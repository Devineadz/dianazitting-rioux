Website going live!

Posted on May 18, 2020

Pushing a Flask based website turned out to be quite challenging on the hosting service I picked, so I stripped down all the Flask elements from my website and decided to go ahead and publish my website with simple HTML, CSS and Bootstrap.

I'm contemplating on either continuing with Flask to get my original website working or first at least to play around with JavaScript to learn to create the same elements through JS instead of using flaskforms and other Flask features.

I think first I will concentrate on is creating a dynamic blog and a contact form using a database. CSV's could be sufficient but I'm itching to put my studies in SQL databases in use!

I'm also excited to add more Bootstrap features and some cool effects, so that is definitely down the list. My website is far from being ready and I will use it as a way to learn more about web development.

After my website is ready, I will continue on my journey on learning more about making games on Unity and next I would like to try to create games on mobile platforms. Despite having no classes during summer, I feel like there just aren't enough hours in a day for everything I'd like to try!
